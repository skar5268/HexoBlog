EJS 樣板語言介紹

發表於 2024-04-28

EJS 全名為 Embedded JavaScript templating，它將 HTML 程式碼分為模板（Template）和動態資料（JSON、其他 .ejs 檔案），以 JS 生成 HTML 程式碼。
換句話說，EJS 使開發者能在 HTML 中嵌入 JS，以渲染內容，在管理網站的共同區塊上非常方便。
